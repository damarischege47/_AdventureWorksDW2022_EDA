# AdventureWorksDW2022 EDA Portfolio

## Overview
This project demonstrates Exploratory Data Analysis (EDA) using SQL on the AdventureWorksDW2022 database.  
The analysis includes customer insights, product analysis, revenue trends, and advanced metrics like RFM segmentation.

## Files in the Repository
- **AdventureWorksDW2022_EDA.sql** → All SQL queries used in the analysis.
- **Screenshots_Customers.zip** → Screenshots for customer analysis.
- **Screenshots_Products.zip** → Screenshots for product analysis.
- **Screenshots_Revenue.zip** → Screenshots for revenue analysis.
- **Screenshots_AdvancedEDA.zip** → Screenshots for advanced EDA (RFM, trends, etc.).

## Screenshots
Screenshots are organized into zip files corresponding to each section:  

1. **Customers** → Insights like total customers, customers by country, and gender.  
2. **Products** → Top-selling products, revenue by category, and product trends.  
3. **Revenue** → Total revenue, revenue by region, and profit analysis.  
4. **AdvancedEDA** → RFM segmentation, cohort analysis, and other advanced SQL insights.

## How to Use
1. Download the repository.  
2. Extract the zip files to view the screenshots.  
3. Open `AdventureWorksDW2022_EDA.sql` in SQL Server Management Studio (SSMS) or any SQL editor to run the queries.  

## Notes
- All analysis is done using SQL only — no external tools.  
- Screenshots illustrate the results of the SQL queries.
